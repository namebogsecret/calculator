## Быстрый старт

### Первый запуск

1. **Откройте калькулятор** в веб-браузере
2. **Выберите режим** в верхней панели:
   - **Основной** - базовые арифметические операции
   - **Научный** - расширенные математические функции
   - **Графики** - построение и анализ графиков функций
   - **Статистика** - статистический анализ данных

3. **Начните вычисления** - введите выражение и нажмите "="

### Интерфейс

```
┌─────────────────────────────────────────────────────────┐
│ [ Основной ] [ Научный ] [ Графики ] [ Статистика ]     │ ← Режимы
├─────────────────────────────────────────────────────────┤
│                    2 + 3 * 4                           │ ← Выражение
│                       14                               │ ← Результат
├─────────────────────────────────────────────────────────┤
│ [sin] [cos] [tan] [log] [ln] [√] [x²] [x³] [π] [e]    │ ← Научные функции
├─────────────────────────────────────────────────────────┤
│ [C] [CE] [⌫] [÷]     [7] [8] [9] [×]                   │
│ [x²] [√] [%] [−]    [4] [5] [6] [+]                    │ ← Кнопки
│ [1/x] [±] [.] [=]   [1] [2] [3] [=]                    │
│                     [0] [.] [=]                        │
└─────────────────────────────────────────────────────────┘
```

## Основной калькулятор

### Базовые операции

#### Арифметические операции
- **Сложение**: `2 + 3 = 5`
- **Вычитание**: `10 - 4 = 6`
- **Умножение**: `7 × 8 = 56` (можно использовать `*`)
- **Деление**: `15 ÷ 3 = 5` (можно использовать `/`)

#### Дополнительные операции
- **Процент**: `200 + 15% = 230`
- **Смена знака**: `±5 = -5`
- **Обратное число**: `1/x для x=4 = 0.25`

### Операции с памятью

#### Кнопки памяти
- **MS** (Memory Store) - сохранить в память
- **MR** (Memory Recall) - извлечь из памяти
- **M+** - добавить к памяти
- **M−** - вычесть из памяти
- **MC** (Memory Clear) - очистить память

#### Пример использования памяти
```
1. Введите: 150
2. Нажмите: MS (сохранить 150 в память)
3. Введите: 25 + 30 = 55
4. Нажмите: M+ (добавить 55 к памяти: 150 + 55 = 205)
5. Нажмите: MR (показать содержимое памяти: 205)
```

### Управление вводом

- **C** - полная очистка (выражение и результат)
- **CE** - очистка текущего ввода
- **⌫** - удаление последнего символа
- **=** - вычисление результата

## Научный калькулятор

### Тригонометрические функции

#### Основные функции
- **sin(x)** - синус
- **cos(x)** - косинус  
- **tan(x)** - тангенс

#### Обратные функции
- **asin(x)** - арксинус
- **acos(x)** - арккосинус
- **atan(x)** - арктангенс

#### Примеры
```
sin(30°) = 0.5
cos(π/3) = 0.5  
tan(45°) = 1
asin(0.5) = 30° (в градусах) или π/6 (в радианах)
```

### Логарифмические функции

- **log(x)** - десятичный логарифм (основание 10)
- **ln(x)** - натуральный логарифм (основание e)
- **log₂(x)** - логарифм по основанию 2

#### Примеры
```
log(100) = 2
ln(e) = 1
log₂(8) = 3
```

### Степенные функции

- **x²** - квадрат числа
- **x³** - куб числа
- **x^y** - возведение в произвольную степень
- **√x** - квадратный корень
- **∛x** - кубический корень
- **ⁿ√x** - корень n-й степени

#### Примеры
```
5² = 25
2³ = 8
2^10 = 1024
√16 = 4
∛27 = 3
```

### Константы

- **π** (пи) = 3.14159265359...
- **e** (число Эйлера) = 2.71828182846...
- **φ** (золотое сечение) = 1.61803398875...

### Статистические функции

- **n!** - факториал числа
- **nPr** - размещения (P(n,r))
- **nCr** - сочетания (C(n,r))

#### Примеры
```
5! = 120
P(10,3) = 720
C(10,3) = 120
```

### Режимы углов

Переключение между градусами и радианами:
- **DEG** - градусы (360° = полный круг)
- **RAD** - радианы (2π = полный круг)

## Построение графиков

### Вкладка "Функции"

#### Построение простых функций

1. **Введите функцию** в поле ввода:
   ```
   x^2                  # Парабола
   sin(x)              # Синусоида
   exp(x)              # Экспонента
   log(x)              # Логарифм
   abs(x)              # Модуль
   ```

2. **Нажмите "Построить"** или Enter

3. **Используйте элементы управления**:
   - Колесо мыши - масштабирование
   - Перетаскивание - перемещение
   - Двойной клик - автомасштабирование

#### Сложные функции

```
x^3 - 3*x^2 + 2*x + 1        # Кубическая функция
sin(x) * cos(x)              # Произведение тригонометрических функций
sqrt(x^2 + 1)                # Корень из суммы
(x^2 - 1)/(x + 1)           # Дробно-рациональная функция
```

#### Построение нескольких графиков

1. Постройте первую функцию
2. Введите вторую функцию
3. Нажмите "Добавить график"
4. Повторите для дополнительных функций

### Вкладка "Таблица"

#### Создание таблицы значений

1. **Введите функцию**: `x^2 + 2*x + 1`
2. **Задайте диапазон**:
   - От: `-5`
   - До: `5`
   - Шаг: `0.5`
3. **Нажмите "Создать таблицу"**

#### Результат
```
┌─────┬─────────┐
│  x  │  f(x)   │
├─────┼─────────┤
│ -5  │   16    │
│ -4.5│  11.25  │
│ -4  │   9     │
│ ... │   ...   │
│  5  │   36    │
└─────┴─────────┘
```

### Вкладка "Регрессия"

#### Анализ данных

1. **Введите данные** (x,y пары):
   ```
   1,2
   2,4
   3,6
   4,8
   5,10
   ```

2. **Выберите тип регрессии**:
   - Линейная (y = ax + b)
   - Квадратичная (y = ax² + bx + c)
   - Экспоненциальная (y = ae^(bx))
   - Логарифмическая (y = a·ln(x) + b)
   - Степенная (y = ax^b)

3. **Получите результаты**:
   ```
   Линейная регрессия:
   y = 2x + 0
   R² = 1.000 (идеальное соответствие)
   ```

### Вкладка "Трассировка"

#### Анализ графика

1. **Постройте функцию**
2. **Включите режим трассировки**
3. **Используйте стрелки** для перемещения по графику
4. **Просматривайте координаты** точек
5. **Находите особые точки**:
   - Максимумы и минимумы
   - Точки пересечения с осями
   - Точки перегиба

## Советы и хитрости

### Эффективное использование

#### 1. Цепочки вычислений
```
Вместо: 2 + 3 = 5, затем 5 * 4 = 20
Используйте: 2 + 3 * 4 = 14 (учитывая приоритет операций)
Или: (2 + 3) * 4 = 20 (с скобками)
```

#### 2. Использование памяти
```
Вычислите: sin²(x) + cos²(x) для x = 30°
1. sin(30) = 0.5 → MS
2. 0.5^2 = 0.25 → M+
3. cos(30) = 0.866...
4. 0.866^2 = 0.75 → M+
5. MR = 1 (тождество sin²(x) + cos²(x) = 1)
```

#### 3. Построение сложных графиков
```
Для изучения y = sin(x) + cos(2x):
1. Постройте y = sin(x) (красный цвет)
2. Добавьте y = cos(2x) (синий цвет)
3. Добавьте y = sin(x) + cos(2x) (зеленый цвет)
4. Сравните результаты
```

### Математические хитрости

#### 1. Быстрое возведение в степень
```
2^10 = 2^8 * 2^2 = 256 * 4 = 1024
3^4 = (3^2)^2 = 9^2 = 81
```

#### 2. Приближенные вычисления
```
√2 ≈ 1.414
π ≈ 22/7 ≈ 3.14286
e ≈ 2.718
```

#### 3. Проверка результатов
```
Для проверки log₁₀(1000) = 3:
10^3 = 1000 ✓

Для проверки sin(90°) = 1:
asin(1) = 90° ✓
```

## Решение проблем

### Частые ошибки

#### 1. "Синтаксическая ошибка"
**Причина**: Неправильно введенное выражение
**Решение**: 
- Проверьте скобки: `((2+3)*4)`
- Используйте правильные символы: `*` вместо `×`
- Избегайте пробелов в числах: `123` вместо `1 23`

#### 2. "Деление на ноль"
**Причина**: Попытка деления на ноль
**Решение**: 
- Проверьте знаменатель
- Для функций типа `1/x` избегайте x = 0

#### 3. "Значение вне области определения"
**Причина**: Недопустимое значение для функции
**Примеры**:
- `log(-1)` - логарифм отрицательного числа
- `sqrt(-1)` - корень из отрицательного числа
- `asin(2)` - арксинус числа > 1

#### 4. "Переполнение"
**Причина**: Результат слишком велик
**Решение**: 
- Используйте более мелкие числа
- Разбейте вычисления на этапы

### Проблемы с графиками

#### 1. График не отображается
**Проверьте**:
- Правильность синтаксиса функции
- Диапазон отображения
- Область определения функции

#### 2. График выглядит неправильно
**Возможные причины**:
- Неподходящий масштаб - используйте автомасштабирование
- Недостаточное количество точек - уменьшите шаг
- Особенности функции (разрывы, асимптоты)

#### 3. Медленная работа с графиками
**Оптимизация**:
- Уменьшите диапазон построения
- Упростите функцию
- Закройте другие вкладки браузера

### Проблемы со статистикой

#### 1. "Недостаточно данных"
**Решение**: Введите минимум 2 значения для большинства вычислений

#### 2. "Некорректные данные"
**Проверьте**:
- Разделители (запятые или пробелы)
- Отсутствие текста в числовых данных
- Корректный формат (1.5 вместо 1,5 в некоторых локалях)

#### 3. Неожиданные результаты регрессии
**Возможные причины**:
- Выбросы в данных
- Неподходящий тип регрессии
- Недостаточно точек данных

### Общие рекомендации

#### 1. Обновите браузер
Убедитесь, что используете современную версию браузера с поддержкой ES6 модулей.

#### 2. Очистите кэш
При проблемах с загрузкой очистите кэш браузера.

#### 3. Проверьте JavaScript
Убедитесь, что JavaScript включен в настройках браузера.

#### 4. Используйте консоль разработчика
Откройте F12 → Console для просмотра ошибок.
