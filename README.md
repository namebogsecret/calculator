# Продвинутый графический калькулятор

Современный многофункциональный калькулятор с научными функциями, построением графиков и статистическим анализом. Проект построен на модульной архитектуре с разделением на компоненты для удобства разработки и поддержки.

## 🚀 Возможности

### Основной калькулятор
- ✅ Базовые арифметические операции (+, -, ×, ÷)
- ✅ Операции с памятью (M+, M-, MR, MC)
- ✅ История вычислений (до 50 записей)
- ✅ Работа с десятичными числами и процентами
- ✅ Предпросмотр результата в реальном времени

### Научный калькулятор
- ✅ Тригонометрические функции (sin, cos, tan, и их обратные)
- ✅ Гиперболические функции (sinh, cosh, tanh)
- ✅ Логарифмические функции (log, ln, log₂)
- ✅ Степенные функции и корни (x², x³, x^y, √, ∛)
- ✅ Математические константы (π, e, φ)
- ✅ Статистические функции (n!, nPr, nCr)
- ✅ Переключение режимов углов (DEG/RAD)

### Построение графиков
- ✅ Построение графиков функций y = f(x)
- ✅ Интерактивное масштабирование (zoom in/out)
- ✅ Панорамирование и навигация
- ✅ Трассировка точек на графике
- ✅ Таблицы значений функций с настраиваемым шагом
- ✅ Добавление нескольких функций на один график
- ✅ Экспорт графиков

### Статистический анализ
- ✅ Описательная статистика (среднее, медиана, дисперсия, квартили)
- ✅ Регрессионный анализ (линейная, квадратичная, экспоненциальная, логарифмическая, степенная)
- ✅ Двумерная статистика и корреляционный анализ
- ✅ Статистические тесты (t-тест, z-тест, хи-квадрат)
- ✅ Работа с распределениями (нормальное, биномиальное, Пуассона)
- ✅ Визуализация данных (гистограммы, диаграммы размаха)
- ✅ Управление списками данных (L1, L2, L3, L4, L5, L6)

### Дополнительные модули
- ✅ Матричные операции (Matrix class)
- ✅ Конвертер единиц измерения (UnitConverter)
- ✅ Финансовые функции (сложные проценты, аннуитеты, амортизация)
- ✅ Распределения вероятности (нормальное, биномиальное, Пуассона, экспоненциальное)

### Современный дизайн
- ✨ Современная цветовая палитра (indigo, cyan, pink)
- ✨ Анимации и эффекты (glow, pulse, shimmer, scan)
- ✨ Glassmorphism с эффектом размытия
- ✨ Адаптивный дизайн для всех устройств
- ✨ Плавные переходы и микро-взаимодействия

## 🛠 Установка и запуск

### Требования
- Современный веб-браузер с поддержкой ES6 модулей
- Локальный веб-сервер (для разработки)

### Запуск проекта

1. **Клонируйте репозиторий:**
   ```bash
   git clone <repository-url>
   cd calculator
   ```

2. **Запустите локальный сервер:**
   
   С помощью Python (если установлен):
   ```bash
   python -m http.server 8000
   ```
   
   С помощью Node.js (если установлен):
   ```bash
   npx serve .
   ```
   
   С помощью VS Code Live Server (расширение)

3. **Откройте в браузере:**
   ```
   http://localhost:8000
   ```

## 🏗 Архитектура

### Модульная структура
Проект использует современную модульную архитектуру ES6 с разделением ответственности:

- **Компоненты** - 7 HTML шаблонов для разных частей интерфейса
- **Стили** - 10 CSS модулей (~18KB), разделенных по функциональности
- **Скрипты** - 9 JavaScript модулей (~3900 строк кода) с четким разделением логики

### Статистика кода
- **JavaScript**: 3904 строки кода
  - app.js (638 строк) - главный модуль приложения
  - statistics.js (715 строк) - статистический анализ
  - ui.js (663 строк) - управление интерфейсом
  - graphing.js (602 строк) - построение графиков
  - scientific.js (490 строк) - научные функции
  - calculator.js (335 строк) - основной калькулятор
  - utils.js (295 строк) - вспомогательные функции
  - constants.js (100 строк) - константы и конфигурация
  - component-loader.js (66 строк) - загрузчик компонентов

### Основные классы

#### `Calculator` (calculator.js)
Основной класс калькулятора с базовыми операциями:
```javascript
const calc = createCalculator();
calc.appendToExpression('2 + 2');
const result = calc.calculate(); // { success: true, result: 4, formattedResult: "4" }
```

#### `GraphRenderer` (graphing.js)
Рендеринг и управление графиками:
```javascript
const renderer = createGraphRenderer(canvas);
renderer.addFunction('x^2');
renderer.zoomIn();
renderer.exportImage();
```

#### `DescriptiveStatistics` (statistics.js)
Статистический анализ данных:
```javascript
const stats = createDescriptiveStatistics([1, 2, 3, 4, 5]);
const summary = stats.summary();
console.log(summary.mean); // 3
console.log(summary.median); // 3
console.log(summary.standardDeviation); // 1.414...
```

#### UI Менеджеры (ui.js)
- **DisplayManager** - управление отображением
- **ModeManager** - переключение режимов
- **TabManager** - управление вкладками
- **InputHandler** - обработка ввода
- **DataListManager** - управление списками данных
- **NotificationManager** - уведомления
- **FormHandler** - обработка форм
- **ResizeHandler** - адаптивность

## 🎨 Кастомизация

### Изменение темы
Цветовая схема определена в `src/styles/variables.css`:
```css
:root {
    --primary: #3b82f6;
    --secondary: #10b981;
    --dark: #0f172a;
    /* ... другие переменные */
}
```

### Добавление новых функций
1. Добавьте функцию в соответствующий модуль (`scientific.js`, `calculator.js`, etc.)
2. Обновите UI в `ui.js` для добавления кнопки
3. Добавьте стили в соответствующий CSS файл

### Создание новых режимов
1. Создайте HTML шаблон в `src/components/`
2. Добавьте CSS стили в `src/styles/`
3. Реализуйте логику в новом JavaScript модуле
4. Зарегистрируйте режим в `ModeManager`

window.calculatorApp
